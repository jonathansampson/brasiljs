"Just My Code"

Internet Explorer now supports code "black-boxing". With "Just My Code" you can mark libraries and 3d-party scripts as Library Code and skip those during debugging. Files having the suffix min.js are assumed to be library code a priori.

Source maps & Source Map Designation

For various reasons, library code may not contain a source map comment in its footer. The F12 Debugger will now allow you to manually associate a map to a given source file. By right-clicking on a file in the debugger, you can now select the Choose source map option.

Event Breakpoints & Tracepoints

You can use Event Breakpoints to interrupt difficult-to-locate events in your page life-cycle. If you're not looking for something that extreme, you can use Event Tracepoints to merely notify you of such events.
